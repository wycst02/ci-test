# RFC 2616 - Hypertext Transfer Protocol -- HTTP/1.1

**Status**: Obsoleted by RFC 7230-7235
**Authors**: R. Fielding, J. Gettys, J. Mogul, H. Frystyk, L. Masinter, P. Leach, T. Berners-Lee
**Date**: June 1999

## Overview

HTTP/1.1 is an application-level protocol for distributed, collaborative, hypermedia information systems. It is a generic, stateless protocol that can be used for many tasks beyond hypertext.

## Key Concepts

### Message Format
```
generic-message = start-line
                   *(message-header CRLF)
                   CRLF
                   [ message-body ]
```

### Request Example
```
GET /index.html HTTP/1.1
Host: www.example.com
Accept: text/html

```

### Response Example
```
HTTP/1.1 200 OK
Content-Type: text/html
Content-Length: 1234

<html>...</html>
```

### Chunked Transfer Coding
```
Chunked-Body   = *chunk
                 last-chunk
                 trailer
                 CRLF

chunk          = chunk-size [ chunk-extension ] CRLF
                 chunk-data CRLF
last-chunk     = 1*("0") [ chunk-extension ] CRLF
```

## Notes

- RFC 2616 has been obsoleted by RFC 7230-7235 (2014)
- HTTP/1.1 requires persistent connections by default
- All HTTP date/time stamps MUST be represented in GMT
- Cache directives defined in Cache-Control header
- Entity tags (ETag) used for cache validation
